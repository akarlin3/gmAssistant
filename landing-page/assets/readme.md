# Gamemaster Assistant Landing Page Assets

To keep this webpage 100% portable, performant, and simple to copy-paste without broken image links, all icons, badges, and logo designs are drawn using inline, responsive **SVG elements** directly within the HTML/CSS code.

This folder can be used to store any additional custom image screenshots, custom SVG logos, or branding assets you want to link.
